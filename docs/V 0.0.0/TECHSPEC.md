## 3. TS-HOME-01: 01번 스텝 설명 문구 변경

### 3.1 배경

기존 문구는 "질문이 OPIC과 유사한 주제 중 완전 무작위로 제공된다"는 인상을 준다. 실제 의도한 사용자 경험은 **사용자가 원하는 주제를 먼저 선택**하고, 그 주제에 속한 콤보 질문 3개를 무작위로 받는 방식이다. 문구가 이 흐름을 반영하도록 수정한다.

### 3.2 변경 내용

| 구분 | 내용 |
|---|---|
| 대상 | `steps[0].description` (`src/pages/Home.tsx`) |
| 기존 | 실제 OPIC과 유사한 주제로 구성된 질문 3개를 무작위로 받아요. |
| 변경 | 자신이 원하는 주제로 구성된 콤보 질문 3개를 무작위로 받아요. |

### 3.3 영향 범위 참고

- `기능명세서.md`의 FR-HOME-02("클릭 시 Interview 페이지로 이동하며 랜덤 Topic+질문 세트 요청 트리거")는 "완전 랜덤 Topic"을 전제로 한다. 본 문구 변경은 **주제 선택 UI**를 전제로 하므로, 실제로 주제 선택 기능이 함께 도입되는 시점에 `기능명세서.md`(FR-HOME-02, `GET /api/questions/random` 파라미터)도 함께 갱신이 필요하다. 본 문서는 Home 화면 문구 변경 범위만 다루며, API/백엔드 변경은 별도 스펙으로 관리한다.

## 4. TS-HOME-02: 스텝 섹션 스크롤 등장 애니메이션

### 4.1 요구 사항

"이렇게 진행돼요" 섹션(01/02/03 카드)에서 사용자가 스크롤을 내릴 때, 카드가 **하나씩 순서대로 나타나며 각 지점에서 잠시 멈추는** 연출을 적용한다.

### 4.2 동작 정의

1. 섹션 진입 전: 01/02/03 카드는 모두 비노출(투명 + 하단 오프셋) 상태로 대기
2. 사용자가 아래로 스크롤하여 섹션에 진입하면, 01 카드가 먼저 페이드인 + 슬라이드업으로 나타나고 잠시 멈춤(스크롤 스냅 또는 딜레이)
3. 스크롤을 더 내리면 02 카드가 같은 방식으로 등장, 다시 03 카드가 등장
4. 세 카드가 모두 나타난 이후에는 일반 스크롤로 복귀(다음 섹션으로 자연스럽게 이동)

### 4.3 구현 가이드

| 항목 | 내용 |
|---|---|
| 트리거 방식 | `IntersectionObserver` 기반 뷰포트 진입 감지 (각 카드별 개별 관찰) |
| "멈춤" 연출 | CSS `scroll-snap-stop: always` + 섹션 컨테이너 `scroll-snap-type: y proximity` 조합, 또는 카드별 순차 딜레이(stagger) 애니메이션으로 대체 가능 |
| 등장 트랜지션 | `opacity 0 → 1`, `translateY(24px) → 0`, `duration 400~500ms`, `ease-out` |
| 순차 지연(stagger) | 카드 간 150~200ms 딜레이 |
| 접근성 | `prefers-reduced-motion: reduce` 사용자는 애니메이션 없이 즉시 표시 |
| 재생 정책 | 최초 1회만 재생(뷰포트 재진입 시 재생 안 함) — 반복 애니메이션으로 인한 어지러움 방지 |

### 4.4 비고

- 스크롤 스냅형 "멈춤" 연출은 트랙패드/모바일 스크롤 관성과 충돌할 수 있으므로, 초기 구현은 stagger 딜레이 방식(스냅 없이 순차 등장)으로 시작하고 사용자 테스트 후 스냅 적용 여부를 재검토한다.

## 5. TS-HOME-03: 자주 묻는 질문(FAQ) 섹션

### 5.1 배치 위치

Home 페이지 최하단, 기존 `footer`(면책 문구 영역) **바로 위**에 신규 섹션으로 추가한다.

### 5.2 구성

| 항목 | 내용 |
|---|---|
| 섹션 제목 | "자주 묻는 질문" |
| 표시 방식 | 아코디언(질문 클릭 시 답변 펼침/접힘), 다른 항목 열어도 기존 항목 유지(다중 오픈 허용) |
| 스타일 | `designsystem.md` §5 컴포넌트 가이드 준수 — `surface` 배경 카드, `radius-lg`, `shadow-card`, 구분선은 `rgba(0,0,0,0.15)` |
| 초기 상태 | 전체 접힘 |

### 5.3 FAQ 항목(초안)

| 순서 | 질문 | 답변 |
|---|---|---|
| 1 | 로그인이 꼭 필요한가요? | 아니요. 회원가입/로그인 없이 바로 연습을 시작할 수 있어요. |
| 2 | 녹음한 음성이 저장되나요? | 아니요. 음성은 텍스트 변환에만 사용되고, 변환 즉시 폐기되며 저장하지 않아요. |
| 3 | 몇 문제를 연습하나요? | 한 세션에 3문제(콤보 질문)를 연습해요. |
| 4 | 결과가 실제 OPIC 성적과 같나요? | AI 기반 추정치로, 실제 OPIC 성적과 차이가 있을 수 있어요. |

> FAQ 문구는 초안이며, 실제 서비스 정책(주제 선택 기능 도입 등) 확정 후 최종 검수가 필요하다.
